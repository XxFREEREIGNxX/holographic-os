## Philosophy

We treat the Universe not as a collection of matter, but as a **computational ledger** where physics is the executive kernel of a holographic operating system.

This project implements a self-consistent toy universe governed by:
- **Directed Transition Principle (DTP)** — Irreversible state evolution via a 4×4 Transition Operator `T^`
- **Nested Completion-Coupled Response Model (NCCR)** — Dynamical completion and debt amortization
- **Fisher-Rao Gradient Flow** — Variational efficiency (the Garbage Collector)
- **Holographic Horizon Dynamics** — Dark Energy as structural debt service
- **Lloyd Bound** — Finite computational lifespan (~214 Gyr)

## Features

- Multi-scale NCCR simulation across topological cycles
- Explicit 4×4 `T^` matrix evolution with completion feedback
- Persistent homology / Betti number mapping to particles
- Standard Model eigenmode interpretation (proton, photon, electron, gluons, etc.)
- Real-time Fisher-Rao optimization minimizing action variance
- Dynamic horizon `L` growth (Dark Energy analog)
- Automatic Lloyd Bound shutdown
- Physics dashboard with proton lifetime, CMB proxy, and cosmic shutdown projection
